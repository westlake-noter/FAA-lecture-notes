### English

This release archives the materials for the **Fundamental Algebra and Analysis** course.

As of June 24, 2026, all course sessions have ended. This repository will now be maintained only very infrequently. Future updates are expected mainly for typo fixes, corrections, or small improvements contributed by readers.

The attached files contain the compiled PDF materials for the course, including lecture notes, exercises, and related handouts when available. The LaTeX source files remain available in the repository for reference.

Please use this release as the primary download location for PDF bundles.

### Included Materials

- Part I lecture notes
- Part II lecture notes
- Exercises and related handouts, if included in the attached archive
- Source files remain in the repository

### Notes

- These materials are provided as-is.
- If you find a typo or error, please open an issue or submit a focused pull request.
- Further maintenance is not planned unless corrections are contributed.
