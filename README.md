# Fundamental Algebra and Analysis

This repository archives the source files for the **Fundamental Algebra and Analysis** course notes.

As of June 24, 2026, all course sessions have ended. The repository is kept mainly as an archive of the LaTeX sources for the two parts of the course:

- `I/`: Part I course notes and related materials
- `II/`: Part II course notes and related materials

Compiled PDFs, including exercises and other handouts when available, will be packaged and published through GitHub Releases. Please check the repository's Releases page for downloadable PDF bundles instead of expecting built PDFs to be committed directly to the source tree.

Maintenance will be very infrequent. Future updates are expected only when someone reports a typo, correction, or small improvement and contributes it directly.

## Contributing

Typo fixes and small corrections are welcome. If you notice an error, please open an issue or submit a pull request with a focused change.

# 代数与分析基础

这个仓库用于归档 **代数与分析基础** 课程讲义的源文件。

截至 2026 年 6 月 24 日，今天所有课程均已结束。本仓库主要作为两部分课程 LaTeX 源文件的归档：

- `I/`：第一部分课程讲义及相关材料
- `II/`：第二部分课程讲义及相关材料

编译后的 PDF，包括习题和其他讲义材料（如有），会打包发布在 GitHub Releases 中。请优先在本仓库的 Releases 页面下载 PDF 包，而不是期待编译后的 PDF 被直接提交到源代码目录中。

之后本仓库的维护频率会非常低。预计只有在有人主动指出 typo、修正错误，或贡献小改进时，才会继续更新。
